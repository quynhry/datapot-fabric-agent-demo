# Security Policy

This is a research / educational repository. It ships **only 100% dummy, AI-generated data** and
is not a production system.

## Reporting a concern
Please email **hoang@datapot.edu.vn** (subject: `SECURITY — datapot-fabric-agent-demo`) rather than
opening a public issue if you are reporting any of:

- A value in the bundled dataset that you believe may be **real or re-identifiable** (a name,
  account, or figure that maps to a real person or institution). We treat this as our
  **highest-priority** report.
- A leaked credential, token, workspace/tenant ID, or connection string committed anywhere in the
  repo or its history.
- Any other vulnerability in the scripts or tooling included here.

We will acknowledge within **5 business days** and aim to remediate confirmed issues promptly.
Please allow a reasonable window before any public disclosure.

## Out of scope
The data is intentionally synthetic and the project is intentionally *not production-tier*
(see [README.md](README.md)). Model accuracy, numeric realism, and report quality are **not**
security issues — please open a normal GitHub issue for those.
