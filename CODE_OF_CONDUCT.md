# Code of Conduct

## Our standard
This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its Code of Conduct. The full, canonical text is available at that link and is incorporated here
by reference.

In short: we are committed to a welcoming, respectful, and harassment-free experience for everyone,
regardless of background or identity. We expect all participants — maintainers, contributors, and
community members — to act professionally and considerately in all project spaces (issues, pull
requests, discussions, and any related communication).

## Scope
This Code of Conduct applies within all project spaces and when an individual is representing the
project in public spaces.

## Reporting
If you experience or witness unacceptable behavior, please report it confidentially to the
maintainers at **hoang@datapot.edu.vn**. All reports will be reviewed and handled discreetly, in
line with the Contributor Covenant's enforcement guidelines.
